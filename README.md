# Modern Animated E-Commerce Website

A full-stack e-commerce project with a modern animated frontend and Express backend.

## Tech Stack

- Frontend: React + Vite + TailwindCSS + Framer Motion + React Router
- Backend: Node.js + Express
- Data: JSON dataset (`data/products.json`)

## Project Structure

- `frontend/` React application
- `backend/` Express API (`/api/products`)
- `data/products.json` product database

## Quick Start

### 1) Install dependencies

```bash
npm run frontend:install
npm run backend:install
```

### 2) Run backend

```bash
npm run backend:dev
```

### 3) Run frontend

```bash
npm run frontend:dev
```

Open `http://localhost:5173`.
